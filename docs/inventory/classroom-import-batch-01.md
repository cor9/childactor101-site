# Classroom Import Batch 01

This is the first classroom import batch plan for the Child Actor 101 learning library.

It does not attempt to import the full inventory.

It prioritizes:

- Evergreen parent confusion topics
- Import-first lessons with minimal reinvention
- Topics that strengthen `/start-here`
- Topics that build trust early
- Topics that reduce scams, wasted money, and unrealistic expectations

## Batch Summary

- Total lessons: 25
- Import-first lessons (`import_as_is_or_lightly_update` or `import_with_update`): 19
- Split/create lessons from saved source docs: 6
- Chapter cap: no more than 4 lessons from any one chapter

## Chapter Balance

- Start Here: 4
- Representation: 4
- Auditions: 4
- Professional Materials: 3
- Parent Operations: 3
- Wellness & Resilience: 2
- Safety & Ethics: 4
- Career Growth: 1

## Selection Notes

- Batch 01 favors lessons that answer first-order parent questions.
- Several saved curriculum sources are included only where the topic gap is too important to delay, especially `How to Become a Child Actor`, `How to Build a Better Audition Slate`, `How to Use a Slate Shot Well`, `How to Track Auditions and Expenses`, and `Maintaining Self-Esteem as a Child Actor`.
- Topics with duplicate clusters were only selected when a canonical lesson is clearly needed in the first batch.
- For module-derived lessons, use `docs/level-up/source-lesson-inventory.md` as the internal lesson-splitting map and do not expose internal module naming publicly.
