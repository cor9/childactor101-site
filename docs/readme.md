# Child Actor 101 Website

## Purpose

This repository contains the complete redesign of ChildActor101.com.

The website serves as the central hub for the Child Actor 101 ecosystem, connecting educational resources, training programs, SaaS tools, community resources, podcasts, blog content, and coaching into one cohesive experience.

This is not simply a marketing website.

It is intended to become the digital headquarters for families navigating the television and film industry.

---

## Primary Goals

• Educate parents

• Build trust

• Improve SEO

• Increase newsletter subscriptions

• Introduce the Child Actor 101 ecosystem

• Drive traffic to specialized products

• Showcase Corey Ralston as an authority
