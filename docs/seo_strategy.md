# SEO Strategy

The purpose of search engine optimization is not simply to generate traffic.

Its purpose is to help parents discover trustworthy educational resources exactly when they need them.

Child Actor 101 should become the definitive educational authority for families navigating professional youth acting.

SEO should always support that mission.

---

# Purpose

This document defines the search strategy for the Child Actor 101 Platform.

It covers:

- Organic Search
- AI Search
- Entity SEO
- Topical Authority
- Technical SEO
- Structured Data
- Internal Linking
- Evergreen Content

SEO decisions should always reinforce the platform's educational mission.

---

# Guiding Philosophy

We do not create pages to rank.

We create exceptional educational resources that deserve to rank.

Search engines reward expertise.

Parents reward trust.

The platform should optimize for both.

---

# SEO Goals

Primary Goals

• Become the leading educational resource for parents of young actors.

• Build topical authority around youth acting.

• Establish Corey Ralston as a recognized industry educator.

• Increase qualified organic traffic.

• Increase newsletter subscribers.

• Connect visitors naturally to the Child Actor 101 ecosystem.

---

# Search Intent

Every page should satisfy one primary search intent.

Examples:

Informational

"How do child actors get agents?"

Educational

"How do self tapes work?"

Transactional

"Acting classes for child actors"

Navigational

"Child Actor 101"

Avoid pages that attempt to satisfy multiple unrelated search intents.
